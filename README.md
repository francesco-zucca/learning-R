# Learning R

This repository tracks my progress in learning R for data analysis. Inside, you will find personal notes, custom exercises, and practice scripts. Because AI is writing a lot of code these days (and that's only going to increase), my focus is to understand very well the basics. That way, I will be able to spot and fix AI mistakes and write clean and optimized code.

## Main references

The core material in this repo will be based on the entirety of the book "R for Data Science (2e)".

##  Repository Structure

The project uses the following repository structure:

├── R-for-data-science/  # Notes and custom exercises for the "R for Data Science" book
├── renv/                # Project-specific R library environments
├── .Rprofile            # R startup script (used to activate the renv environment)
├── .gitignore           # Gitignore rules
├── README.md            # Project overview and details
├── learning-R.Rproj     # RStudio project configuration file
└── renv.lock            # Exact dependency versions for reproducibility
